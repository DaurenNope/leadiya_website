# Leadiya

B2B lead intelligence platform for Kazakhstan. Find companies with verified director contacts, phones, and tender data.

## Features

- 12,000+ companies from Kazakhstan with verified contacts
- Filter by city, industry, budget, and ICP score
- Real-time tender data integration
- Export to CSV, Excel, CRM integrations (amoCRM, Bitrix24)
- API access for developers

## Tech Stack

- React + TypeScript
- Vite
- CSS (custom)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output in `dist/` folder.

## Deploy

Connect to Vercel:
- Build command: `npm run build`
- Output directory: `dist`
